# Public schedule site package

This folder can be uploaded directly to a static website hosting service.

Required files:

- index.html
- app.js
- styles.css
- schedule-data.js
- schedule.json

Suggested hosting:

1. Create a public static site with Netlify Drop, Cloudflare Pages, or GitHub Pages.
2. Upload the contents of this public_site folder, or upload public_site.zip.
3. After you get the public URL, turn it into a QR code for staff phones.

Notice: this package contains staff names, shifts, units, and leave/off information.
