# 人員網頁版班表

這個資料夾是人員手機或電腦可閱讀的班表網站。

## 檔案用途

- `index.html`：網頁主畫面。
- `schedule-data.js`：給瀏覽器直接讀取的班表資料。
- `schedule.json`：同一份班表資料的 JSON 版本。
- `app.js`：月份切換、搜尋、班表顯示與列印功能。
- `styles.css`：網頁樣式。

## 顯示規則

- 只顯示已公告或已確認的班表。
- 目前放入 `115/01` 到 `115/06` 的確定歷史班表。
- `115/07` 之後若尚未正式公告，網頁不顯示草稿，只顯示尚未公告。
- 網頁不顯示人員的 `行政` 或 `臨床` 角色字眼。
- `of`、`VO`、`公假` 等不上臨床班的日期以紅色系標示。
- 統計時數時，`of/休假` 不算上班；`VO` 與 `公假` 算入上班時數。

## 更新方式

1. 在排班系統完成正式公告。
2. 從根目錄開 `排班系統入口.cmd`。
3. 按 `人員網頁上線`。
4. 系統會更新 `site`，並產生可上傳的 `public_site` 與 `public_site.zip`。

## 上傳方式

可把 `05_網站發布/public_site` 資料夾或 `05_網站發布/public_site.zip` 上傳到 Netlify Drop、Cloudflare Pages 或 GitHub Pages 等靜態網站服務。
